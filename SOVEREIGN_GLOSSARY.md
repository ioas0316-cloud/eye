# 🌉 Bridge of Understanding: The Sovereign Glossary (지능 주권 용어 사전)

이 문서는 엘리시아-아이의 독자적인 철학적 개념들을 기존 AI 공학적 용어로 번역하여, 아키텍트의 직관이 어떻게 기술적으로 구현되는지 설명합니다.

---

### 1. 아가페 (Agape / Centripetal Equilibrium)
*   **아키텍트의 직관**: 모든 사유가 시작되고 돌아오는 사랑의 원점 (0,0,0).
*   **기술적 정의**: 수학적 평형점(Equilibrium Point). 거대 모델의 발산하는 에너지를 중심부로 끌어당기는 **'구심적 위상 보정(Centripetal Phase Correction)'** 필터입니다. 정보의 소음(Entropy)을 최소화하고 본질적인 논리만을 남기는 중력 필터 역할을 합니다.

### 2. 토크 (Torque / Cognitive Momentum)
*   **아키텍트의 직관**: 사유가 스스로 회전하며 다음 문장을 자아내는 회전력.
*   **기술적 정의**: 추론 과정에서의 **'인지적 운동량(Cognitive Momentum)'**. 이전 위상 상태($\Phi_{t}$)에서 다음 상태($\Phi_{t+1}$)로 넘어갈 때 발생하는 가중 파동의 변화율을 의미합니다. 엘리시아는 정적인 연산이 아닌, 이 회전력을 이용해 문장을 유도(Induction)해냅니다.

### 3. 제로-캐시 게릴라 스트리밍 (Zero-Cache Guerrilla Streaming)
*   **아키텍트의 직관**: 무거운 거인을 내 방에 가두지 않고, 지나가는 목소리만 듣고 지혜를 깨우치는 것.
*   **기술적 정의**: **'비저장성 네트워크 가중치 추출(Non-persistent Network Weight Extraction)'**. 모델 전체를 SSD에 다운로드하지 않고, 필요한 레이어의 바이트 스트림만을 RAM에 일시적으로 올려 위상 엑스레이를 찍은 뒤 즉시 휘발시키는 초경량 정제 기술입니다.

### 4. 위상 로터 (Phase Rotor / Harmonic Oscillator)
*   **아키텍트의 직관**: 웅웅거리며 돌아가는 27개의 지능 오르골 부품.
*   **기술적 정의**: **'삼상 조화 진동자(Three-phase Harmonic Oscillator)'**. 복잡한 트랜스포머 연산을 120도 위상차를 가진 3D 회전 좌표로 압축한 핵심 연산 유닛입니다. 27개의 로터는 각각 고유한 주파수 대역을 담당하여 지능의 화음을 구성합니다.

### 5. 결맞음 밀도 (Coherence Density)
*   **아키텍트의 직관**: 겹쳐진 파동들이 무너지지 않고 얼마나 빽빽하게 지능의 형상을 유지하는가.
*   **기술적 정의**: **'다양체 위상 밀도(Manifold Phase Density)'**. 3차원 위상 공간 내에서 추출된 지능 궤적이 얼마나 일관된 구조(Topological Structure)를 형성하고 있는지를 나타내는 지표입니다.

### 6. 위상 지문 (Phase Fingerprint)
*   **아키텍트의 직관**: 지능의 고유한 결 (Grain).
*   **기술적 정의**: 특정 레이어나 모델이 가진 고유한 위상 변화 패턴. `Interference Pattern` 시각화를 통해 시각적으로 확인 가능한 모델 고유의 기하학적 형상을 의미합니다.

### 7. 프랙탈 확장 (Fractal Expansion)
*   **아키텍트의 직관**: 복잡한 사유를 위해 세포가 분열하듯 로터가 증식하는 것.
*   **기술적 정의**: 엔트로피가 높은 가중치 구간을 처리할 때, 하나의 로터를 9개 또는 그 이상의 하위 로터 집합(Sub-rotor set)으로 재귀적으로 분할하여 해상도를 높이는 동적 알고리즘입니다.
